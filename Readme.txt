Tutor Management System
The Tutor Management System is a web-based application designed to streamline the interaction between tutors and students, making the process of finding, communicating with, and managing tutoring services efficient and user-friendly. This system aims to bridge the gap between students seeking knowledge and tutors offering their expertise.

Key Features
Student Portal
The student portal provides a seamless interface for students to search for tutors based on various criteria such as name, country, city, and subject expertise. Students can view detailed profiles of tutors, including their qualifications, experience, and contact information. Additionally, students can manage their profiles, view their tutoring sessions, and communicate directly with tutors.

Tutor Portal
Tutors have their dedicated portal where they can manage their profiles, including adding their qualifications, experience, and available subjects. They can also view a list of students who have shown interest in their services, manage their tutoring sessions, and communicate with students directly through the platform.

Search and Filter Functionality
A powerful search and filter functionality allows both students and tutors to find exactly what they need quickly. Students can search for tutors by name, location, subject expertise, and more. Tutors can search for students by various criteria to tailor their offerings better.

Secure Authentication
The system ensures that both students and tutors have secure access to their respective portals. The authentication mechanism includes session management to ensure that only authenticated users can access their dashboards and relevant information.

Admin Dashboard
An admin dashboard is included to oversee the entire operation of the platform. Admins can manage users, monitor activities, and generate reports to ensure the smooth functioning of the system.

Technologies Used
ASP.NET: The backbone of the application, providing a robust framework for web development.
C#: The programming language used for server-side logic.
SQL Server: Database management to handle data storage and retrieval efficiently.
Bootstrap: Front-end framework to ensure a responsive and user-friendly interface.
HTML/CSS: For structuring and styling the web pages.

Installation and Setup
1) Clone the repository:  git clone https://github.com/yourusername/TutorManagementSystem.git
2) Navigate to the project directory: cd TutorManagementSystem
3) Set up the database: Restore the SQL Server database using the provided script in the Database folder.
4) Update the connection string: In the web.config file, update the connection string to point to your SQL Server instance.
5) Build and run the project: Open the solution in Visual Studio, build the project, and run it.
